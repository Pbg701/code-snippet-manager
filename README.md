Code Snippet Manager (Python Tkinter Compiler)
This project is a desktop-based Python code editor and compiler built using Python Tkinter. It allows
users to write, run, save, and manage Python code snippets through a graphical interface.
Features:
- Python code editor with syntax highlighting
- Run Python code and view output/errors
- Open, save, and manage files
- File and folder explorer
- Dark and Light mode support
- Auto documentation generated from comments
- GitHub upload using GitHub API
Technologies Used:
- Python 3
- Tkinter
- tkcode
- subprocess
- requests
- os
How to Run This Project:
1. Install Python 3 on your system.
Check installation using: python --version
2. Install required libraries using pip:
pip install tkcode requests
3. Navigate to the project folder and run:
python main.py
Purpose:
This project is created for learning Python GUI development, file handling, subprocess execution,
and GitHub API integration.
